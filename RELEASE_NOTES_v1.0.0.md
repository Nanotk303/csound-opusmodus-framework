# Release v1.0.0 — Initial Public Release

This first public release introduces the Csound–Opusmodus Framework as a unified
Lisp DSL for algorithmic composition and Csound-based synthesis.

## Included

- Core framework (`src/Csound.lisp`)
- Instrument library (`src/CsoundInstrumentsLib.lisp`)
- Full PDF manual (`docs/manual.pdf`)
- Starter examples (`examples/`)

## Highlights

- Structured instrument definitions with `defcsinstr`
- Named event generation through `cs-event`
- Automatic routing and `.csd` generation
- Real-time and offline rendering workflows

## Notes

Update example score paths as needed for your local system before rendering.
